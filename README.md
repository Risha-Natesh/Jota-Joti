# Prakruthi Shaale

This is the official website for Prakruthi Shaale, a flagship initiative by Outdoor Dynamix. It aims to revolutionize education by connecting students with nature through innovative outdoor learning experiences.

## About the Project

This website is built to showcase the vision, mission, and programs offered by Prakruthi Shaale. It provides information about their various initiatives, including programs for youth, corporate partnerships, and community volunteering.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [ShadCN UI](https://ui.shadcn.com/)
- **Language**: TypeScript

## Getting Started

To run this project locally, follow these steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
